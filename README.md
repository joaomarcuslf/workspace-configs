# workspace-configs

Default configs

## Getting Started

1. Copy ```sample.env``` to ```.env``` and rename the variables if you need
2. Build the images and run the containers:

```sh
make docker-start
```

You can stop the docker by running: `make docker-stop`

## Running Tests

```sh
make test
```
